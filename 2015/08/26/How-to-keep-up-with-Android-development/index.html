如何紧跟Android开发的潮流

发表于 2015-08-26   |   分类于 翻译   |  

本文翻译来自原文链接

今天看了篇文章，开发如何跟上发展的潮流，跟我的感受不谋而合，故整理翻译一下，以飨他人。

android在世界上发展的非常迅猛，作为一个android 开发者，很重要的一件事就是要跟上前沿潮流，每次主要的发布都带来一堆新特性，工具不断改善，每周都有新lib的不断诞生，每天都有不错的博客帖子发布，阅览，尝试，精通所有这些东西是不现实的也不是最终目标–专注于筛选出来的简明优质的资源和渠道就好。以下是我如何筛选的一些小tips：

Twitter

在我看来，推特是让我保持在前沿的最重要的渠道，通过关注正确的人，可以让你的推特成为获取开发相关消息和课题的源头。你在推特上可以找到google的伙计们和非常棒的android开发者，以下是我关注的一些id(应该是都需要翻墙)：

  • google官方账户， Android developers accounts
  • Square Engineering , Square 工程师官方账号，开源过大量有用的库。

。。。。。
好吧，原文还有好多大牛的推特账号，但是加链接好麻烦，就不一一挂过来了，感兴趣的可以去我的Twitter看看我关注的大神即可，我关注的基本都是android开发者，除了个别NBA star 哈哈，
我的Twitter地址
。。。。。
另外，要随时浏览你关注的推特的推文哦，并且，为了最小化冗余信息的干扰，随时对不活跃和太多废话的账号取消关注
如果你不玩Twitter，试一下google+，上面的大多数人在google+也同样活跃。

youtube

youtube是学习android开发乃至其他的软件开发的很棒的内容资源，拥有DevBytes series, and the recent Android Performance Patterns videos等系列视频的Google’s Android Developers channel是最值得关注的，还有 Google developers 和 Square Engineering ，都有很棒的关于android但不仅限于手机系统的演讲。

博客和网站

有很多值得关注的关于android开发的博客

  • Google’s official blog
  • Square Engineering Blog
  • android ui ux
  • Dan Lew’s blog
  • the Square Island blog 这个是媛媛
  • Styling Android 提升app ui ux技术上指导的网站
  • Grokking Android 研究android开发细节

也可以订阅以下网站，每周会发布android相关的新鲜文章

  • androidweekly
  • ANDROID DEV DIGEST
  • androiddev 非常赞的帖子和讨论

经常阅览新闻网站，随时了解开发界发生的新鲜事。关注生产厂家定制的UI的截图能让你提升对android的理解。（Also, seeing all those screenshots about manufacturer UI customisations will make you appreciate stock Android even more.）
即便过滤掉了不感兴趣的帖子，依然有很多要看的。每天阅读文章的时候，使用网址收藏，书签服务（比如Pocket，谷歌浏览器的一个插件）是很有帮助的，

多读变更日志

与时俱进？阅读changes！不论是 android studio，Gradle ，github，构建工具，第三方库，只要你使用它，那就去了解他，This helps finding out if there were breaking changes introduced in the update.
如果你在项目里用一些第三方库，在github上follow他们。这样你就第一时间获得变更通知，如果你每天（on a daily basis）都用他们，你同样可以把变更随时加入项目中。

学术会议和聚会（Conferences and meetups）

开发者大会是个学习新技术的很棒的地方，可以见到很多有意思的同行。你可以从这里，这里，还有这里 找。当然了还有Google I/O，只要你足够幸运列席。更常见的是，即使你不能参加，也可以去网上找一下演示文稿和视频资料。

其他杂项Miscellaneous

  • 加入google的android兴趣组
  • google+ Android Development ,Android Wear Developers
  • android后台
  • 播客，有声杂志哦podcast

这些与时俱进的秘诀不是让你作为额外负担一样的去阅览，把它包含在每天的计划中：早餐的时候浏览浏览Twitter，坐公交时候看看博客，把看肥皂剧的时间用来看youtube的演讲视频。
