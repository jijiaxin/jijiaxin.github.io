Java泛型 && Fiddler抓包

2015-08-18

java中的泛型

泛型程序设计，意味着编写的代码可以被很多不同类型的对象所重用。例如，我们并不希望为聚集String和File对象分别设计不同的类。实际上，也不需要这样做，因为一个ArrayList类可以聚集任何类型的对象。这是一个泛型程序设计的实例。

在java SE 7及以后的版本中，构造函数中可以省略泛型类型：ArrayList files = new ArrayList<>();省略的类型可以从变量的类型推断得出。
编译器也可以很好地利用这个信息，当调用get的时候，不需要进行强制类型转换，编译器就知道返回类型是String，而不是object。add的时候编译器可以检查，避免插入错误类型的对象，类型不符会无法编译通过，这比运行时出现类的强转异常要好的多。
泛型的魅力在于，使程序具有更好的可读性和安全性。

类型变量使用大写形式，在java库中，

  • 用变量E表示集合的元素类型
  • K和V分别表示表的关键字与值的类型
  • T(需要时还可以用临近的字母U和S)表示“任意类型”

Android转场动画的实现

2015-08-17

要想提高android用户体验，转场动画是必不可少的一个耍酷技能
android转场动画可分为windowAnimation和ActivityAnimation，见名知意，一个是window级，一个是Activity级别

WindowAnimation 与 ActivityAnimation 比较

主要区别有：

  • WindowAnimation包括 windowEnterAnimation 和 windowExitAnimation ；
    ActivityAnimation包括 activityOpenEnterAnimation， activityOpenExitAnimation ， activityCloseEnterAnimation 和 activityCloseExitAnimation
  • 在项目中WindowAnimation优先级大于Activity，即在Activity转场过程中，如果同时设置了WindowAnimation和ActivityAnimation,那么很可能只会执行WindowAnimation
  • 对于WindowAnimation的定义很简单，在style.xml文件中只需要继承Animation Style即可

REST设计风格的理解

2015-08-17

REST，即Representational State Transfer 表现层状态转化
是Fielding（ HTTP规范的主要编写者之一）提出并定义的一套对互联网软件的架构原则，REST是一种设计风格而不是标准
我们可以根据这些原则设计以系统资源为中心的 Web 服务
如果你在设计应用程序时能坚持REST原则，那就预示着你将会得到一个使用了优质Web架构（这将让你受益）的系统。

资源表现层

“资源”是一种信息实体，它可以有多种外在表现形式。”资源”具体呈现出来的形式，叫做它的”表现层”（Representation）。
比如，文本可以用txt格式表现，也可以用HTML格式、XML格式、JSON格式表现，甚至可以采用二进制格式；图片可以用JPG格式表现，也可以用PNG格式表现。

URI只代表资源的实体，不代表它的形式。严格地说，有些网址最后的”.html”后缀名是不必要的，因为这个后缀名表示格式，属于”表现层”范畴，而URI应该只代表”资源”的位置。它的具体表现形式，应该在HTTP请求的头信息中用Accept和Content-Type字段指定，这两个字段才是对”表现层”的描述。

REST中的资源所指的不是数据，而是数据和表现形式的组合，比如“最新访问的10位会员”和“最活跃的10位会员”在数据上可能有重叠或者完全相同，而由于他们的表现形式不同，所以被归为不同的资源，这也就是为什么REST的全名是Representational State Transfer的原因。

Markdown格式体验

2015-08-09

markdown使用总结

（我为了方便观看对比，对于示例，第一组是原文格式，故意没加空格，第二组是显示效果，自然要加空格）

标题格式

标题前加#，一级标题加一个#，二级标题加##，以此类推，总共有六级标题，#与文字要加空格，否则不能区别
另外也有三个以上- - -或者===表示标题

列表

无序列表 只要在每列前加个 - 即可 不要忘记符号与文字之间加空格

-衣带渐宽终不悔
-为伊消得人憔悴

  • 为伊消得人憔悴
  • 衣带渐宽终不悔

或者加号+，星号*都可以 ，表现出来都是原点的效果，但是最好用-或者+，星号容易与加粗斜体冲突

  • 鱼沈雁杳天涯路，始信人间别离苦
  • 诚知此恨人人有，贫贱夫妻百事哀
