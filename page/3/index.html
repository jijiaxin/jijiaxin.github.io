DLNA调研

2015-11-19

DLNA的全称是DIGITAL LIVING NETWORK ALLIANCE(数字生活网络联盟)， 其宗旨是Enjoy your music, photos and videos, anywhere anytime， DLNA(Digital Living Network Alliance) 由索尼、英特尔、微软等发起成立、旨在解决个人PC，消费电器，移动设备在内的无线网络和有线网络的互联互通，使得数字媒体和内容服务的无限制的共享和增长成为可能，目前成员公司已达280多家。

分类

DLNA将其整个应用规定成5个功能组件。从下到上依次为：网络互连，网络协议，媒体传输，设备的发现控制和管理，媒体格式。
功能模块：DMS　DMR　DMC　DMP
DMS：Digital Media Server 服务器
DMR：Digital Media Renderer 渲染器（受控端）
DMC：Digital Media Controller 控制器
DMP：Digital MediaPlayer 播放器（可理解为DMC+DMR）

生活随笔（叁-国庆中）

2015-10-05

云在青山月在天，人间有味是清欢。

国庆小长假就要over了，小惆怅呀。。。在家天天慵懒度日，晒晒太阳，睡个懒觉，吃吃肉，一天嗖，就过去了。

在家什么都好，有一点受不了就是————天天被催婚。。。。
也难怪，在乡下，与我同龄的大多孩子都能打酱油了。我还没媳妇，父母着急我很能理解
而且，买房这件事，已经成了父母心头的一座大山，压着父母，我很心疼。
我也想赶紧找个媳妇，安稳下来，买个房，每月给地主交交按揭。把父母心头的大山搬走，但是。。。
结婚偏又是件急不得的事，马马虎虎随便找个人凑合一下总不是什么好事吧。
唉，为啥我不是个帅哥呢，焦作人。。。

那句话怎么说来着，叫“你越缺什么，就越炫耀什么”！
还有一句话，叫“你越缺什么，就越觉得别人在炫耀什么！”
曾经我觉得第一句话是对的，后来某个瞬间我意识到后面那句才是正确答案。

白发渔樵江渚上，惯看春月秋风。一壶浊酒喜相逢。古今多少事，都付笑谈中。

记得当时年纪小，我爱谈天你爱笑，有一回并肩坐在桃树下，风在林梢鸟在叫。我们不知怎样困觉了，梦里花儿知多少。

人生如白驹过隙，我们不能总是跟风惆怅，我们应该珍惜当下，珍惜身边人。

人生恰如三月花，倾我一生一世念。 来如飞花散似烟，醉里不知年华限。

人生如梦，聚散分离， 朝如春花幕凋零，几许相聚， 几许分离，缘来缘去岂随心。 青丝白发转眼间，漠然回首，几许沧桑在心头。 独自泪空流。
